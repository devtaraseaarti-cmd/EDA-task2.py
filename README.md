# 📊 Exploratory Data Analysis (EDA) – Student Productivity Dataset

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the *Student Productivity Dataset*.
The goal is to understand patterns in student study habits, productivity, and exam performance using Python.

EDA helps in:

* Understanding dataset structure
* Identifying patterns and relationships
* Visualizing important insights through graphs

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* VS Code

---

## 📂 Dataset

Dataset used in this project: **Ultimate Student Productivity Dataset (5000 rows)**

The dataset contains information such as:

* Study hours
* Sleep hours
* Social media usage
* Productivity score
* Exam score

---

## 🔎 Steps Performed

1. Imported required libraries.
2. Loaded dataset using Pandas.
3. Checked dataset structure using:

   * `head()`
   * `info()`
   * `describe()`
4. Checked missing values.
5. Analyzed correlation between numeric columns.
6. Created data visualizations.

---

## 📊 Visualizations

The following graphs were created using Matplotlib:

* Scatter Plot – Relationship between study hours and exam score
* Line Graph – Trend analysis
* Bar Graph – Category comparison
* Pie Chart – Distribution of categories

These visualizations help in understanding how student habits affect performance.

---

## ▶️ How to Run the Project

1. Install required libraries:

pip install pandas matplotlib

2. Run the Python file:

python EDA.py

---

## 📈 Key Insights

* Students who study more hours generally score better in exams.
* Excessive social media usage may reduce productivity.
* Proper sleep helps improve productivity levels.
